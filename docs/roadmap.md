# Roadmap

## V1: 工具库骨架

- 建立 `tools/` monorepo 结构。
- 每个工具拥有独立 README、docs、prompts、examples 和 src。
- 明确不提交敏感数据、真实用户数据和未授权模型。

## V2: 可运行 Demo

- Edge Vision Lab 接入浏览器本地识物。
- Elevator Dispatch Simulator 增加基础仿真环境。
- Industrial Agent Bus 定义消息 schema 和示例。

## V3: 工业智能实验链路

- 视觉结果、仿真状态、智能体消息进入统一工程状态判断流程。
- 增加工业标签库、项目模板和复盘文档。
- 形成可展示、可教学、可二次开发的工业智能工具资产库。
