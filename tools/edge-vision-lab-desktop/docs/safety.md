# Safety Boundary

本工具为 Windows 离线 Demo，仅用于固定工业场景物体识别实验。

## Explicit Non-Goals

- 不做人脸识别。
- 不做身份识别。
- 不做情绪识别。
- 不做人员监控。
- 不做考勤管理。
- 不识别车牌、身份证、银行卡、学生证等敏感信息。
- 不收集真实用户数据。
- 不上传、不保存摄像头画面、图片或视频。

## User Notice

请勿将摄像头对准他人、隐私区域、公共人群场景、车牌、证件、学生信息或其他敏感信息。

识别结果仅用于技术展示和工程学习，不作为生产控制、安防、执法、考勤或商业决策依据。

## Public Release Checklist

- 没有 `.env`。
- 没有 API Key。
- 没有服务器密码。
- 没有数据库账号。
- 没有真实用户数据。
- 没有真实摄像头录像。
- 没有包含人脸、车牌、学生信息或证件的测试图片。
- 没有未授权模型或数据集。
- 没有自动更新、遥测统计、云 API 或账号系统。
