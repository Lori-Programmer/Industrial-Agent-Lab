# Privacy Note
## 隐私说明

## Default Behavior / 默认行为

1. Camera frames are processed in the user's local browser by default.
2. V1.0 does not upload images or videos.
3. V1.0 does not save images or videos.
4. The feedback form only shows a local demo status and does not submit data to a server.

1. 本项目默认在用户本地浏览器中处理摄像头画面。
2. 第一版不上传图片或视频。
3. 第一版不保存图片或视频。
4. 反馈表单只显示本地 Demo 状态，不提交到服务器。

## Explicit Non-Goals / 明确不做

- No face recognition.
- No identity recognition.
- No emotion detection.
- No personnel monitoring.
- No attendance management.
- No age, gender, expression, fatigue, pressure, or personnel-state detection.
- No license plate, ID card, bank card, student card, or other sensitive information recognition.

- 不做人脸识别。
- 不做身份识别。
- 不做情绪识别。
- 不做人员监控。
- 不做考勤管理。
- 不识别年龄、性别、表情、疲劳、压力等人员状态。
- 不识别车牌、身份证、银行卡、学生证等敏感信息。

## User Safety Notice / 用户使用提示

Do not point the camera at other people, private areas, public crowd scenes, license plates, identity documents, student information, or any sensitive personal information.

请勿将摄像头对准他人、隐私区域、公共人群场景、车牌、证件、学生信息或其他敏感个人信息。

## Future Data Features / 未来数据功能

If future versions add image upload, backend models, cloud APIs, user accounts, feedback storage, analytics, or commercial services, the project must add:

- A clear privacy policy.
- User consent before data processing.
- Data deletion mechanism.
- Security protection controls.
- Model and dataset authorization records.
- Public release review.

如果未来接入上传识别、后端模型、云 API、用户账号、反馈存储、数据统计或商业化服务，必须增加隐私政策、用户同意、数据删除、安全保护、模型/数据授权证明和公开发布审核。
