# Elevator Dispatch Simulator

## 这个工具解决什么问题

电梯调度仿真器用于研究多楼层、多电梯、多乘客请求下的调度策略。它面向自动化、智能制造和多智能体学习场景，可用于比较最近车、最短等待、能耗优先、强化学习和 MARL 策略。

## 当前状态

Planned。当前目录是工具骨架，尚未接入真实仿真引擎。

cd tools/elevator-dispatch-simulator

## 安全边界

- 仅用于调度算法学习、仿真和作品集展示。
- 不接入真实电梯控制系统。
- 不作为生产控制、安全决策或人员管理依据。
- 不收集真实人员轨迹、身份、考勤或监控数据。
